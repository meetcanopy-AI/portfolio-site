# Portfolio Site

Static portfolio site for meetcanopy.club.

## Pages

- `index.html`
- `work.html`
- `services.html`
- `about.html`
- `contact.html`
- `thanks.html`

## Deploy

This is a static Netlify site with no build step.

Netlify settings:

- Build command: leave blank
- Publish directory: `.`

The contact form uses Netlify Forms through `contact.html`.
